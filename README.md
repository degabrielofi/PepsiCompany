## Sobre o Projeto

Página web que apresenta os principais refrigerantes da marca [Pepsi](https://www.facebook.com/PepsiBrasil), com design responsivo e visual inspirado na identidade da marca. Cada produto é exibido com imagem e breve descrição.

O projeto foi desenvolvido como exercício de layout responsivo, trabalhando posicionamento, flexbox e estilização com CSS puro.

> Projeto desenvolvido como parte da jornada de aprendizado em desenvolvimento front-end.

## Como Instalar e Rodar

```bash
# Clonar o repositório
git clone https://github.com/degabrielofi/PepsiCompany

# Entrar na pasta
cd PepsiCompany

# Abrir no navegador
start index.html
```
